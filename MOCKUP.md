# Mockup Formats Guide

I can understand mockups in several formats! Here are the formats I can work with:

## 1. **Image Files** (Most Common)
Send me screenshots or mockup images in formats like:
- PNG, JPG, JPEG
- SVG
- WebP
- GIF

**Example:**
```
"Here's my mockup: /path/to/mockup.png"
```

## 2. **ASCII/Text Art** (Quick sketches)
```
┌─────────────────────────────┐
│  Header                     │
├─────────────────────────────┤
│ [Image]  | Product Name     │
│          | Price: 100,000₫  │
│          | [Add to Cart]    │
└─────────────────────────────┘
```

## 3. **Structured Text Description**
```
Layout:
- Top: Product image (300x300px)
- Middle:
  * Product title (bold, 18px)
  * Price (red, 16px)
  * Description (gray, 14px)
- Bottom: Add to cart button (full width, green)
```

## 4. **HTML/CSS Wireframe**
```html
<div class="product-card">
  <img src="..." />
  <h3>Product Title</h3>
  <p class="price">100,000₫</p>
  <button>Add to Cart</button>
</div>
```

## 5. **Figma/Design Tool Links**
- Figma public links
- Adobe XD share links
- Sketch Cloud links

## 6. **Reference to Existing Pages**
"Make it look like the homepage but with [specific changes]"
